# 📊 FinSight

A Personal Financial Intelligence Dashboard built with Python and Power BI to transform raw financial transactions into meaningful business insights through data cleaning, feature engineering, fraud analysis, and interactive visualizations

---


📖 Overview

FinSight is an end-to-end data analytics project that demonstrates the complete analytics workflow—from raw data to business intelligence. The project focuses on cleaning and transforming financial transaction data, performing exploratory analysis, engineering meaningful features, and presenting actionable insights through an interactive Power BI dashboard.

Rather than simply visualizing data, FinSight emphasizes understanding transaction behavior, identifying fraud patterns, analyzing spending trends, and presenting key performance indicators that support data-driven decision making.

---

 ✨ Project Highlights

 Cleaned and transformed raw transaction data using Python and Pandas
 Performed feature engineering to generate meaningful analytical variables
 Built key performance indicators (KPIs) for executive reporting
 Analyzed fraud patterns across different locations and transaction types
 Explored monthly trends and peak transaction hours
 Designed a professional interactive dashboard in Power BI
 Presented insights using modern data visualization techniques
